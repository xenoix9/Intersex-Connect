# Intersex Connect Website

A comprehensive website for intersex individuals to find support, resources, and community.

## Features

- Personal stories and experiences from intersex individuals
- Resources for healthcare, mental health, and legal rights
- Community forums and events calendar
- Advocacy information and campaigns
- Responsive design for all devices

## File Structure

- `index.html` - Home page
- `about.html` - About us page
- `stories.html` - Personal stories section
- `resources.html` - Resources and support information
- `community.html` - Community hub with forums and events
- `advocacy.html` - Advocacy and activism section
- `css/` - Stylesheets
  - `style.css` - Main styles
  - `responsive.css` - Responsive styles
- `js/` - JavaScript files
  - `main.js` - Common functionality
  - `stories.js` - Stories page functionality
  - `community.js` - Community page functionality
- `images/` - Website images

## How to Use

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Navigate through the different sections using the menu

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome icons
